      由于本平台为纯静态部署（GitHub Pages），暂不支持在线上传。


      📌 贡献流程：

      1. 将您编写好的 HTML 动效文件 Push 至项目的 GitHub / GitLab 仓库的 /effects 目录下；

      2. 在仓库根目录的 platform.html 中的 FX_DATA 数组里，新增一条资产数据（名称、分类、文件路径）；

      3. 提交 Pull Request，合并后 CI/CD 将自动构建并刷新本页面的资产列表。


      💡 命名建议：effects/my-awesome-fx.html

      🏷️ 分类可选：环境与氛围 / UI与交互 / 卡牌专属 / 材质与遮罩
